Blog of Blog

Blog, a blog generated by blog

You can also set author, date as above.

blog

the most creative named static blog generator

You can use $title, $date, $author, $blog_title, $blog_desc, $blog_url or any variable exported in config

Config

Basic properties

You must create a config file named "config" in your blogdir

blog_title="Blog's Name"
blog_desc="Blog's description"
blog_url="https://blogs/url"
blog_author="Default Author"

Markup syntax

We use smu as the default markup language. If you want to use another one, you can adjust it in the config file

For instance,

markup="pandoc"
markup_link="https://pandoc.org/"

or

markup="lowdown"
markup_link="https://kristaps.bsd.lv/lowdown/"

But I didn't do a lot of tests on anything other than smu.

Directories

The standared directories are

blogdir=$(pwd)
templ="templ"
posts="posts"
public="public"

Templates

Templates are in $blogdir/$templ.

For now there are three template files

  • post_template.html

template for posts in $blogdir/$posts. This template can't access $postlist variable.

  • index_template.html

your first page, the $post variable is filled by parsing the index.md file.

  • rss_template.xml

used to generate rss.xml

Usage

Just run blog in your source directory.

This directory, $blogdir, must contain the config file, and the Directories $templ, $posts and $public.

The index.md file also must be in $blogdir.

BE CAREFUL

The config file is just a bash! If you write the wrong code you can screw it up, like rewriting cleanf and deleting its source code.

But the config file is just a bash! You can rewrite the functions for a particular use. Or add something that makes sense to a project but not everyone.
